Ocuhub

September - December 2014

Eye gaze menu to be used with Eyetribe to control opening of applications completely with the eyes.
When reading the menu and an icon is about to be selected, a timer/loading wheel will appear around
the icon to show when the application will be selected. Also multiple menu pages can be flipped
through and there are submenus for larger topics such as games.
